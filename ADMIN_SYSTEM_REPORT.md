#### 默认管理员账号

1. **SQL脚本创建** - 使用提供的 `create_admin_account.sql` 脚本
2. **数据库直接插入** - 在 `sys_user` 表中插入管理员记录
3. **角色权限** - role=1 表示管理员角色，status=0 表示正常状态
